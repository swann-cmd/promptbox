## 🚀 部署指南

### 快速部署（3 步）

#### 选项 1：使用 Vercel CLI（推荐）

```bash
# 1. 登录 Vercel（首次使用）
npx vercel login

# 2. 部署到生产环境
npx vercel --prod

# 3. 等待部署完成，访问提供的 URL
```

#### 选项 2：使用部署脚本

```bash
# 运行自动部署脚本
./deploy.sh
```

#### 选项 3：通过 Vercel 网站（最简单）

1. 访问 [vercel.com/new](https://vercel.com/new)
2. 导入 GitHub 仓库：`swann-cmd/promptbox`
3. Vercel 自动检测 Vite + React 配置
4. 点击 "Deploy" 按钮
5. 等待 2-3 分钟，部署完成！
